# Análise Exploratória do ENEM 2023

Este repositório contém uma análise estatística e exploratória dos microdados do ENEM 2023, focada em identificar disparidades socioeconômicas e padrões de desempenho entre diferentes perfis de candidatos.

## 🚀 Objetivo do Projeto
O objetivo principal é aplicar técnicas de Ciência de Dados para responder perguntas críticas sobre o cenário educacional brasileiro, utilizando bibliotecas como **Pandas**, **Seaborn** e **Matplotlib**.

## 📂 Fonte de Dados
Os dados utilizados nesta análise foram extraídos do dataset **Student Performance**, disponível no Kaggle através do link:
[Kaggle - Student Performance (ENEM 2023)](https://www.kaggle.com/datasets/jpamcb/student-performance/data)

## 📊 Perguntas Respondidas
1. Qual o desempenho médio por região do Brasil?
2. Alunos de escola pública vs privada: qual a diferença média em Matemática?
3. Qual o impacto da renda familiar na nota média?
4. Existe diferença de desempenho entre homens e mulheres por área?
5. Qual a distribuição de notas em Matemática? (Histograma)
6. Estados do Nordeste vs resto do Brasil: como se comparam?
7. Ter internet em casa faz diferença nas notas?
8. Qual a correlação entre nota de Redação e nota de Matemática?

## 🛠️ Tecnologias Utilizadas
* **Python 3.12+**
* **Pandas**: Manipulação de dados tabulares.
* **Seaborn & Matplotlib**: Visualização de dados estatísticos.
* **Jupyter Notebook**: Ambiente de desenvolvimento e documentação.

## 📂 Estrutura do Repositório
* `notebooks/`: Contém o arquivo `01_exploracao.ipynb` com as análises.
* `data/`: Pasta destinada ao dataset (não inclusa no commit por restrições de tamanho).
* `requirements.txt`: Lista de dependências do projeto.
